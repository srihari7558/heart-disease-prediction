# ❤️ Heart Disease Prediction App

## 📌 Project Overview

This project predicts the risk of heart disease using a Machine Learning model (Logistic Regression).

## ⚙️ Technologies Used

* Python
* Streamlit
* Scikit-learn
* NumPy

## 📊 Model Details

* Algorithm: Logistic Regression
* Accuracy: 84%
* Features: Age, Cholesterol, Blood Pressure, Smoking, etc.

## 🚀 How to Run

1. Clone the repository
2. Install dependencies

pip install -r requirements.txt

3. Run the app

streamlit run app.py

## 📈 Features

* User-friendly interface
* Real-time prediction
* Risk probability display
